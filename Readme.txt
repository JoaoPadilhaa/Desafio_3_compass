📌 eCommerce Fullstack (Em Desenvolvimento)

Aplicação web de eCommerce desenvolvida com React (frontend) e Node.js (backend), com integração completa entre interface, API e banco de dados, baseada em um layout do Figma.

O sistema permite a visualização de produtos e categorias, navegação entre páginas, exibição de detalhes e interação com funcionalidades como formulário de contato.

🚧 Funcionalidades (em desenvolvimento)

Listagem de produtos e categorias via backend

Página inicial com destaques de produtos

Navegação por categorias

Página de detalhes do produto

Paginação e navegação entre produtos

Integração completa entre frontend e backend

Formulário de contato com persistência em banco de dados

🛠️ Tecnologias utilizadas

React

TypeScript

Node.js / NestJS

Banco de dados relacional

ORM (ex: Sequelize)

Integração com API REST

📎 Observações

Projeto fullstack com foco em boas práticas como Clean Code, arquitetura organizada e uso de metodologias ágeis (Scrum). Baseado em protótipo do Figma, com implementação das principais páginas de um eCommerce real.
